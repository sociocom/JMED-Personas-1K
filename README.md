# データセット概要 (Dataset Overview)

PP1k は、日本語の合成患者ペルソナJSONLデータセットです。公開統計・補助リソースから構成した確率分布と制約条件をもとに、合成患者の基本属性、疾患情報、検査・治療情報、生活背景、生活習慣、性格特性、ペルソナ記述を生成しています。

本データセットは、医療AI/ヘルスケアLLMの研究・評価、プロンプト検証、デモ用途での利用を想定しています。実在患者の診療録、問診票、検査結果、個人情報を収集または変換したものではありません。

各JSONLは1行が1人分の合成患者レコードです。本READMEでは、`data/persona_demo_100_jp_disease_catalog_gpu01.jsonl` を含む生成済みJSONLを説明対象とします。

## 使用モデル

患者ペルソナ本文の生成には、[GPT-OSS-120B / openai/gpt-oss-120b](https://huggingface.co/openai/gpt-oss-120b) を使用しています。年齢、性別、地域、疾患、生活習慣、疾患カタログ由来の固定条件を与え、JSON Schema に沿う合成患者レコードを生成しています。

## 生成に用いた資料

PP1k は、以下の資料・リソースから作成した確率分布と制約条件をもとに生成しています。

- [厚生労働省 令和5年患者調査](https://www.mhlw.go.jp/toukei/saikin/hw/kanja/23/index.html): 年齢層、性別、傷病分類の分布制約に使用。
- [e-Stat 令和5年患者調査 全国編 Z16-1](https://www.e-stat.go.jp/stat-search/files?cycle=7&layout=datalist&page=1&stat_infid=000040234267&tclass1=000001224322&tclass2val=0&toukei=00450022&tstat=000001224321): 傷病小分類、施設種類、入院/外来別の推計患者数に使用。
- [e-Stat 令和5年国民健康・栄養調査](https://www.e-stat.go.jp/stat-search/files?cycle=7&layout=datalist&tclass1=000001228532&tclass2val=0&toukei=00450171&tstat=000001041744): 喫煙歴、飲酒歴、運動習慣、睡眠、身体計測の制約に使用。
- [日本郵便 UTF-8郵便番号データ](https://www.post.japanpost.jp/service/search/zipcode/download/utf/zip/utf_ken_all.zip): `resources/japan_postal_codes_by_prefecture.json` の都道府県別郵便番号候補に使用。
- [nvidia/Nemotron-Personas-Japan](https://huggingface.co/datasets/nvidia/Nemotron-Personas-Japan): `resources/nemotron_japan_names.json` の日本語姓名候補に使用。姓名は独立した患者名フィールドではなく、ペルソナ文章の主語として使用。
- `resources/prefecture_distribution_targets.json`: 都道府県割付用のローカル補助リソース。現在は47都道府県を同一重みで扱うプレースホルダーであり、外部URLはありません。
- `jp_disease_catalog`: Z16-1から作成したローカル疾患カタログ。主病名、ICDコード、診療科、年齢範囲、性別制約、診療設定、代表症状、代表検査の固定条件に使用。

## データセット詳細 (Dataset Details)

JSONLの各行には、主に次の項目カテゴリが含まれます。

- 基本属性: `患者ID`, `都道府県`, `郵便番号`, `年齢`, `性別`
- 医学情報: `傷病分類`, `主病名`, `ICDコード`, `診療科`, `入院／外来`, `症状`, `検査`, `処方薬`, `治療方針・治療目標`
- 生活背景: `同居／独居`, `生活支援・介護資源`, `居住環境`, `医療アクセス・通院手段`, `ADL/IADL`
- 生活習慣: `喫煙歴`, `飲酒歴`, `運動習慣`, `普段の食生活`, `睡眠`
- 心理・性格: `BIG FIVE性格特性`, `趣味・大切にしている活動`, `患者の語り/代表発話`
- ペルソナ文章: `ペルソナ_基本属性`, `ペルソナ_医学的背景`, `ペルソナ_受診に至った経過`, `ペルソナ_診察・検査所見`, `ペルソナ_治療・服薬上の課題`, `ペルソナ_生活背景・支援状況`, `ペルソナ_価値観・心理面`, `ペルソナ_生活習慣`, `ペルソナ_支援ポイント`, `ペルソナ_統合記述`
